POST /modules/{moduleID}/complete-progress

Mark a module as completed for the current user, unlocks the next module, and calculates the total score from completed activities. Returns both progress and score. 

Request:
{
  "completed": true
}

Response

{
  "id": 1,
  "name": "Python Basics",
  "description": "Introduction to Python programming",
  "background_image": "assets/modules/python-basics.jpg",
  "course_id": 1,
  "user_progress": {
    "locked": false,
    "completed": true,
    "last_accessed": "2025-07-31T15:38:17"
  },
  "score": {
    "total_score": 175.5,
    "completed_at": "2025-07-31T15:38:17"
  }
}
  • completed: Set to true to mark the module as completed.
  • user_progress: Shows the user's progress for this module.
  • score: (Only present if completed) The total score from all completed activities in this module.
  • Automatically unlocks the next module for the user.
  • Creates or updates the score in the student_scores table.
